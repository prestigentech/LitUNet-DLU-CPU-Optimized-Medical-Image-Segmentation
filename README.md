# LitUNet-DLU

This repository contains the official implementation of the paper:

**CPU-Optimized LitUNet for Medical Image Segmentation with a Trainable Dynamic Linear Unit (DLU)**

## Overview

This work investigates lightweight medical image segmentation using a CPU-optimized LitUNet architecture enhanced with a trainable Dynamic Linear Unit (DLU) activation mechanism.

The proposed framework is evaluated on:

- Wound Segmentation Dataset
- Kvasir-SEG Polyp Segmentation Dataset
- Lung CT Lesion Segmentation Dataset

## Key Features

- Lightweight LitUNet architecture
- Trainable Dynamic Linear Unit (DLU)
- CPU and GPU benchmarking
- Memory efficiency analysis
- TensorFlow/Keras implementation
- Reproducible medical image segmentation experiments
